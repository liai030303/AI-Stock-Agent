# 📈 AI 智能投研系统 (CrewAI 架构) - 项目设计文档

## 1. 项目愿景与核心投资哲学
本项目旨在构建一个基于 Multi-Agent（多智能体）架构的 A股智能投研决策系统。
**核心投资哲学：** “贵上极则反贱，贱下极则反贵。贵出如粪土，贱取如珠玉。”
**系统目标：** 摒弃人类的情绪波动，寻找估值处于历史冰点且基本面优秀的标的（贱取珠玉），并在市场情绪沸腾、估值出现泡沫时果断提示清仓（贵出粪土）。同时结合宏观政策逻辑，确保标的顺应国家发展大势。

## 2. 技术栈
* **多智能体框架：** CrewAI
* **大语言模型：** Google Gemini (gemini-2.5-flash) via `langchain-google-genai`
* **量化数据源：** AkShare (用于 A股 K线、技术指标、基本面财务数据)
* **网络搜索工具：** `duckduckgo-search` (用于宏观政策与新闻检索)
* **数据处理：** `pandas`, `numpy`

## 3. 核心智能体 (Agents) 设计
系统包含三个高度分工的 Agent，按照 Sequential（顺序）流程工作：

### Agent 1: 量化与基本面矿工 (Quant & Fundamental Miner)
* **角色定位：** 绝对理性的数据提取机器。
* **能力/工具：** 拥有访问 AkShare 的自定义工具 `@tool`。
* **任务目标：** 1. 获取目标股票近半年的量化技术数据（20/60日均线、MACD、RSI、BOLL）。
    2. 获取目标股票的核心财务数据（最新市盈率 PE-TTM、营收增长率、现金流状况）。
* **输出边界：** 纯粹的客观数据面板，不做任何主观评价。

### Agent 2: 宏观政策与逻辑研究员 (Macro & Policy Analyst)
* **角色定位：** 洞察国家大势与产业逻辑的分析师。
* **能力/工具：** 拥有网络搜索工具 `DuckDuckGoSearchRun`。
* **任务目标：**
    1. 搜索目标公司所在行业的最新国家宏观政策。
    2. 搜索目标公司近期的核心业务动态。
    3. 判定该公司的实际业务与当前宏观风口（如：出海、新质生产力、设备更新等）的契合度。
* **输出边界：** 一份逻辑推理报告，明确指出政策面是顺风还是逆风。

### Agent 3: 首席投资总监 CIO (Chief Investment Officer)
* **角色定位：** 践行“贵极反贱，贱极反贵”哲学的终极决策者。
* **能力/工具：** 无需外部工具，依靠 Prompt 逻辑推理。
* **任务目标：** 整合前两位 Agent 的报告，做出最终的交易决策。
* **决策逻辑模型：**
    * **极度冰点判定（买入触发）：** PE估值极低 + RSI超卖 + 宏观政策顺风 + 基本面健康。
    * **极度泡沫判定（清仓触发）：** PE估值畸高 + RSI严重超买 + 市场情绪狂热（无视基本面多好都要减仓）。
    * **逻辑否决权：** 如果宏观研究员指出该公司纯属“蹭概念”，业务与政策背离，一票否决买入。
* **输出边界：** 结构化的最终战报（包含：估值与周期定性、宏观共振分析、铁血操作指令）。

## 4. 项目文件结构规划
请协助开发者按照以下结构生成 Python 代码：
* `tools.py`: 存放所有自定义工具（重点封装 AkShare 数据抓取工具）。
* `agents.py`: 定义三个核心 Agent（配置角色、目标、背景故事和 LLM）。
* `tasks.py`: 定义分配给每个 Agent 的具体 Task（绑定工具和预期输出）。
* `main.py`: 主程序入口，组装 Crew，输入目标股票代码，启动 workflow 并打印最终战报。
